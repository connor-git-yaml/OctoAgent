## 四、M2 准入结论

**M2 准入声明**：M1.5 验收全部通过，M2 开发可启动。

M2 启动前建议注意以下事项：

1. **F013 新增 fixture**：`full_integration_app` + `full_client` 是专为 F013 场景 A/D 创建的（含 TaskRunner），后续 M2 测试如需验证 Orchestrator 事件链，可直接复用这两个 fixture。

3. **capfire 可观测断言**：当前测试环境设置 `LOGFIRE_SEND_TO_LOGFIRE=false`，logfire 不初始化，span 为空是正确的降级行为。若 M2 需要验证真实 span，需在 fixture 中启用 logfire 并传入 `LOGFIRE_TOKEN`。

4. **测试覆盖率基线**：M1.5 验收后整体覆盖率为 66%，M2 新增模块应将覆盖率维持在此基线或以上。
